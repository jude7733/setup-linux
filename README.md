# setup-linux

Bash script to setup ubuntu based distros (includes multimedia, developer softwares, pip packages and tools for gaming in linux)

<hr>

## HOW TO RUN

`chmod +x *.sh`
to give executable permission to all scripts

`./main.sh`
to execute script
